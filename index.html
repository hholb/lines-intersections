Lines and Intersections

  1. 1. Click inside the canvas to add a point.
  2. 2. Click inside the canvas again to add a second point. A line will connect the points.
  3. 3. Repeat steps one and two to add another line.
  4. 4. Click the 'Find Intersections' and the intersection point will be calculated and a circle will highlight the intersections.

Canvas

Please use a browser that supports HTML Canvas elements.

Description

Overview

The purpose of this project is to get some practice with the math for computing lines and intersections using linear algebra.
